# RIRGRAN

Estudo do react + redux + immutable

## Instalação

### 1 - Criando projeto
```
$ npm init
$ npm install --save react react-dom redux react-redux immutable
$ npm install --save-dev webpack babel-loader babel-core babel-preset-es2015 babel-preset-react
```

### 2 - Rodando projeto
```
$ npm run start
```
